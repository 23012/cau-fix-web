## 1. 프로젝트 개요

### 목적

직원이 시설 하자 및 파손을 신고하면, 담당자가 **접수 → 처리 → 완료** 흐름으로 민원을 처리할 수 있도록 하는 시설관리팀 전용 신고 관리 시스템입니다.

### 사용자 유형

| 역할 | 대상 |
|------|------|
| 신고자 | 직원 |
| 처리자 | 현장직 |
| 관리자 | 시설관리팀 |

### 기술 스택

| 항목 | 내용 |
|------|------|
| 프레임워크 | React 19 + React Router 7 |
| 인증 | JWT 기반 인증 *(미정)* + 관리자 승인 회원가입 |

## 3. 민원 처리 흐름

```
접수전  →  접수  →  진행중  →  완료
```

### 상태별 권한

| 전환 | 처리 주체 | 조건 |
|------|-----------|------|
| 접수전 → 접수 | 처리자 | 민원 내역 상태 직접 변경 |
| 접수 → 진행중 | 처리자 | 처리 시작 시 직접 변경 |
| 진행중 → 완료 | 처리자 + 담당자 최종 승인 *(미정)* | 처리 내역(사진, 글) 입력 후 승인 필요 |

## 4. 화면별 기능 명세

### 📋 민원 대시보드 — `/complain-dashboard/:member-id`

> 🚧 개발 중

민원 목록을 조회하는 페이지 *(상세 기획 미정)*

**역할별 조회 범위**

| 역할 | 조회 범위 |
|------|-----------|
| 신고자 | 본인이 접수한 민원만 조회 가능 |
| 처리자 | 본인이 속한 부서 민원만 조회 가능 |
| 관리자 | 전체 민원 조회 가능 |

**주요 기능**
- 전체 민원 목록 조회
- 기간 / 상태 / 분류 필터링
- 민원 제목 검색
- 번호순 / 최신순 / 오래된순 / 상태순 정렬

---

### 🔍 민원 및 처리 상세 — `/complain-detail/:complain-id`

민원 및 처리 상세 내용을 열람하는 페이지

**[민원 내용]**
- 제목, 부서, 분류, 장소, 접수 시간, 내용(글, 사진) 포함
- 민원 상태별 수정/삭제 권한

  | 상태 | 수정 | 삭제 |
  |------|------|------|
  | 접수전 | ✅ | ✅ |
  | 접수 / 진행중 / 완료 | ❌ | ❌ (관리자 문의) |

- 처리자: 민원 상태 수정 가능
  > ⚠️ 상태를 **완료**로 변경 시, 처리 내역 입력 필수

**[처리 내용]**
- 처리자, 처리 시간, 내용(글, 사진) 포함
- 처리자 클릭 시 이름 / 소속 / 전화번호 확인 가능

---

### ✏️ 민원 접수 — `/complain-write/:complain-id`

신고자가 민원을 작성하는 페이지

**주요 기능**
- 제목, 구분, 장소, 접수 내용 입력
- 접수 일자 및 시간 자동 입력
- 사진 최대 10장 등록 *(용량 및 업로드 방식 미정)*
  > 📱 모바일: 사진 촬영 후 즉시 업로드 가능

---

### 🛠️ 처리 입력 — `/complain-handle/:complain-id`

> 🚧 개발 중

처리자가 민원 처리 완료 내용을 작성하는 페이지

**처리 흐름**

```
민원 상세  →  접수 변경  →  내 보관함에 넣기  →  진행중으로 변경  →  처리 입력(완료)
```

**주요 기능**
- 처리 내용 입력
- 처리 일자 및 시간 자동 입력
- 사진 최대 10장 등록 *(용량 및 업로드 방식 미정)*
  > 📱 모바일: 사진 촬영 후 즉시 업로드 가능

---

### 👥 회원 관리 대시보드 — `/member-dashboard/:member-id`

> 🚧 개발 중

관리자가 전체 회원 목록을 조회하는 페이지

---

### 🔔 알림 — `/alarm-list/:push-id`

민원 진행 현황을 푸시 알림으로 제공하는 페이지 *(방식 미정)*

**주요 기능**
- 알림 클릭 시 상세 내용 열람
  - 💻 데스크톱: 팝업창으로 표시
  - 📱 모바일: 민원 상세 페이지로 이동
- 역할별 알림 범위 차이

  | 역할 | 알림 범위 |
  |------|-----------|
  | 신고자 | 본인이 접수한 민원 현황만 실시간 파악 |
  | 처리자 | 모든 신고자의 민원 현황 전부 실시간 파악 |

---

### 📢 공지사항 — `/notice/:member-id`

관리자가 업로드한 공지사항 목록을 조회하는 페이지

**주요 기능**
- 목록에 카테고리 / 제목 / 업로드 일자 표시
- 카테고리별 공지사항 필터 조회 *(카테고리 분류 미정)*
- 공지사항 제목 검색
- 공지사항 클릭 시 세부 내용 열람
- 관리자: 공지사항 업로드 / 수정 / 삭제 가능

---

### 👤 내 정보 관리 — `/my-info/:member-id`

회원가입 시 입력한 정보를 확인 및 수정하는 페이지

> 💻 데스크톱: 팝업창 / 📱 모바일: 전체 페이지로 이동

**주요 기능**
- 회원 정보 수정 (비밀번호, 전화번호)
- Push 알림 설정 (on / off)
- 로그아웃
  > ⚠️ 모바일에서만 내 정보 페이지 내 포함. 데스크톱은 상단바에서 분리
